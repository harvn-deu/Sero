# Sero Kitchen

Roti homemade dengan cinta untuk orang tua dan keluarga  
Website e-commerce roti modern, order langsung masuk WhatsApp.

Fitur:
- Hero cantik + animasi
- Katalog roti (Manis / Asin / Spesial)
- Keranjang belanja
- Checkout → langsung kirim pesanan ke WA
- Admin panel sederhana (password: sero2025)
- Supabase backend (gratis
- Deploy di Vercel (gratis selamanya)

Tech: Next.js 14 App Router • TypeScript • Tailwind • Shadcn/ui • Framer Motion • Supabase • Zustand
